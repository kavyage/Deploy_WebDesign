# Quick steps to build and deploy webdesign

# Design Tools:
    ## Code - HTML/CSS/JS
    ## NOCode using AI tools with prompt - openai chatgpt, google services, figma, canva

# Deploy Tools:
    ## GIT:
        1. prepare the git repo with design files
        2. goto repo settings > pages on left menu
        4. in build and deployment section
        5. select source deploy from a branch
        6. select branch (main) & select path (/root) and save
        7. click visit site once it is deployed and  live
    
    ## Netlify
        1. prepare required .html files for the design and make it as folder
        2. zip the folder
        3. goto nelify and signin
        4. drop zip file
        5. click the generated url
   
    ## Vercel
        1. prepare the git repo with required .html files and make visibility public
        2. goto vercel and sigin
        3. create new project and integrate with git and deploy
        4. click the generated url
